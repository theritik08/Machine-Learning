# Machine Learning

A collection of machine learning notebooks covering data preprocessing, exploratory data analysis, model building, and evaluation — built and iterated on in Google Colab.

## 📁 Contents

| Notebook | Description |
|---|---|
| `WORKING_WITH_CSV_FILES.ipynb` | Reading, cleaning, and structuring tabular/CSV data using pandas |

## 🛠️ Tech Stack

- Python
- Jupyter / Google Colab
- pandas, NumPy
- scikit-learn
- Matplotlib / Seaborn

## 🚀 Getting Started

Clone the repo:

```bash
git clone https://github.com/theritik08/Machine-Learning.git
cd Machine-Learning
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Open any notebook in Jupyter or Google Colab to run it.
